# Da Vinci Studio Website - স্টাইল গাইড (Style Guide)

এই ডকুমেন্টটিতে প্রজেক্টের ডিজাইন সিস্টেম, কালার স্কিম, টাইপোগ্রাফি, স্পেসিং এবং টেইলউইন্ড সিএসএস (Tailwind CSS v4) কনভেনশনগুলো বিস্তারিত আলোচনা করা হয়েছে। নতুন কোনো পেজ বা কম্পোনেন্ট তৈরি করার সময় এই গাইডলাইনটি মেনে চলতে হবে যাতে সাইটের ভিজ্যুয়াল কনসিস্টেন্সি (Visual Consistency) বজায় থাকে।

---

## ১. টাইপোগ্রাফি (Typography)

প্রজেক্টে দুটি মূল ফন্ট ফ্যামিলি এবং বেশ কিছু কাস্টম হেডিং ও প্যারাগ্রাফ ইউটিলিটি ব্যবহার করা হয়েছে। এগুলো `src/app/globals.css` ফাইলে সংজ্ঞায়িত করা আছে।

### ক. ফন্ট ফ্যামিলি (Font Families)
- **Proxima Nova (`font-proxima`):** এটি প্রধানত হেডিং (Headings), নেভিগেশন লিংক, সোশ্যাল বাটন এবং বড় ডিসপ্লে টেক্সটের জন্য ব্যবহৃত হয়।
- **Montserrat (`font-montserrat`):** এটি ডেসক্রিপশন, প্যারাগ্রাফ এবং সেকেন্ডারি টেক্সটের জন্য ব্যবহৃত হয়।

### খ. কাস্টম টাইপোগ্রাফি ক্লাস (Custom Typography Utilities)
সরাসরি নির্দিষ্ট সাইজ ব্যবহারের পরিবর্তে সিএসএস ভ্যারিয়েবল দ্বারা নিয়ন্ত্রিত কাস্টম টাইপোগ্রাফি ক্লাসগুলো ব্যবহার করা বাঞ্ছনীয়:

| ক্লাস (Class) | ফন্ট সাইজ (Font Size) | লাইন হাইট (Line Height) | ফন্ট ওয়েট (Font Weight) | ব্যবহার (Usage Area) |
| :--- | :--- | :--- | :--- | :--- |
| `h1` | 110px | 100px | Semibold (600) | লার্জ ব্যানার / হিরো টেক্সট |
| `h2` | 80px | 80px | Bold (700) | সেকশন হেডার (যেমন: Blog & News) |
| `h3` | 46px | 54px | Semibold (600) | মাঝারি হেডিং / কার্ডের টাইটেল |
| `h4` | 32px | 38px | Semibold (600) | সাব-সেকশন টাইটেল |
| `h5` | 24px | 32px | Semibold (600) | ছোট কার্ড হেডিং |
| `h6` | 20px | 28px | Semibold (600) | অতি ক্ষুদ্র হেডার |
| `p1` | 18px | 28px | Regular (400) | স্ট্যান্ডার্ড প্যারাগ্রাফ / বিবরণ |
| `p2` | 16px | 24px | Regular (400) | ছোট বডি টেক্সট / ফুটার টেক্সট |
| `btn` | 18px | 20px | Semibold (600) | বাটন ও লিংক টেক্সট |

## ২. কালার প্যালেট (Color Palette)

সাইটটির কালার প্যালেট মূলত ডার্ক এবং প্রিমিয়াম ভাইব দেওয়ার জন্য ডিজাইন করা হয়েছে। কালারগুলো টেইলউইন্ডের কাস্টম কালার ভ্যারিয়েবল হিসেবে রেজিস্টার করা আছে:

| কালার টোকেন (Color Token) | হেক্স কোড (Hex Code) | ব্যবহার (Usage) | উদাহরণ (Tailwind Class) |
| :--- | :--- | :--- | :--- |
| `primary-color` | `#010101` | মূল ব্যাকগ্রাউন্ড কালার | `bg-primary-color` |
| `recording-red` | `#F60100` | ব্র্যান্ড প্রাইমারী রঙ, হোভার, বাটন হাইলাইট | `text-recording-red`, `bg-recording-red` |
| `secondary-dark` | `#464646` | সেকেন্ডারি বাটন ও কার্ড বর্ডার/ব্যাকগ্রাউন্ড | `bg-secondary-dark`, `border-secondary-dark` |
| `white-color` | `#F4F4EA` | ডার্ক ব্যাকগ্রাউন্ডের উপর রিডেবল সাদা টেক্সট | `text-white-color` |
| `body-color` | `#909090` | সেকেন্ডারি ডেসক্রিপশন এবং বডি টেক্সট | `text-body-color` |
| `border-color` | `#DADADA` | সাধারণ বর্ডার বা সেপারেটর লাইন | `border-border-color` |

## ৩. লেআউট এবং গ্রিড (Layout & Grid)

সাইটটিকে কনসিস্টেন্ট রাখার জন্য ডিফল্ট গ্রিড এবং ম্যাক্স-উইডথ লেআউট ব্যবহার করতে হবে:

- **কনটেইনার ক্লাস (`container`):** 
  ডিফল্ট কনটেইনার সাইজ হলো `max-w-[1320px]`, যা মাঝখানে থাকে (`mx-auto`) এবং দুই পাশে ১৬px প্যাডিং দেয় (`px-4`)।
  ```tsx
  <div className="container mx-auto px-4">
      {/* আপনার কন্টেন্ট এখানে থাকবে */}
  </div>
  ```
- **গ্রিড সিস্টেম:**
  কার্ড গ্রিডের জন্য ৩টি কলাম বিশিষ্ট লেআউট ব্যবহার করা হয়েছে (যেমন ব্লগ সেকশনে):
  ```tsx
  <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-8">
      {/* কার্ড এলিমেন্টসমূহ */}
  </div>
  ```

## ৪. বাটন ডিজাইন স্টাইল (Button Styling Guidelines)

বাটন স্টাইল করার সময় নিচের কনভেনশনগুলো অনুসরণ করতে হবে:

1. **সোশ্যাল বাটন (সোশ্যাল লিংক বাটন):**
   গোলাকার পিল-আকৃতির বাটন এবং ভেতরে হোভার এফেক্টের জন্য `relative` এবং `overflow-hidden` পজিশনিং করা থাকে।
   ```tsx
   <Link className="social-btn bg-secondary-dark relative overflow-hidden flex items-center gap-2 px-6 py-4 rounded-[999px]">
       <div className="red-bg absolute inset-0 bg-recording-red"></div>
       <span className="relative z-10">BUTTON TEXT</span>
   </Link>
   ```
2. **সার্কুলার অ্যাকশন বাটন (যেমন: Start the Journey):**
   বৃত্তাকার ঘূর্ণায়মান বা বড় অ্যাকশন বাটন যা স্ক্রলের সাথে মোশন তৈরি করে।
   ```tsx
   <Link className="w-32 h-32 md:w-40 md:h-40 rounded-full bg-secondary-dark flex flex-col items-center justify-center">
       {/* আইকন এবং টেক্সট */}
   </Link>
   ```

## ৫. এনিমেশন ও ইন্টারঅ্যাকশন স্টাইল (Animation Classes)

কাস্টম সিএসএস এনিমেশনগুলো সরাসরি রিইউজেবল টেইলউইন্ড ইউটিলিটি ক্লাস হিসেবে ইমপ্লিমেন্ট করা আছে:

- **`animate-shimmer`:** টেক্সটে চকচকে মেটালিক শিমার ইফেক্ট দেয়।
- **`animate-fade-up`:** স্ক্রিনে আসার সময় এলিমেন্ট নিচ থেকে হালকা ফেইড-ইন হয়ে ওপরে ওঠে।
- **`animate-aurora`:** অরোরা আলোর মতো ব্যাকগ্রাউন্ড গ্লো ইফেক্ট দেয়।
- **`animate-float-slow`:** এলিমেন্টকে মসৃণভাবে ওপরে-নিচে ফ্লোটিং বা ভাসমান দেখায়।

## ৬. কোডিং এবং ক্লাস কনভেনশন (Coding Conventions)

1. **মোবাইল-ফার্স্ট ডিজাইন (Mobile-First Design):**
   সব সময় রেসপনসিভ স্টাইলিং করার সময় বেস ক্লাসগুলো মোবাইলের জন্য লিখবেন এবং এরপর `md:`, `lg:`, `xl:`, `2xl:` প্রিফিক্স ব্যবহার করে স্ক্রিন সাইজ বাড়ার সাথে সাথে পজিশন বা লেআউট পরিবর্তন করবেন।
2. **ক্লাস মার্জিং (`cn(...)`):**
   ডাইনামিক বা কন্ডিশনাল ক্লাস জয়েন করার জন্য সরাসরি স্ট্রিং কনক্যাটিনেশন না করে `@/lib/utils` থেকে `cn()` ফাংশন ব্যবহার করুন। এটি ডুপ্লিকেট ক্লাস ওভাররাইট করতে সাহায্য করে:
   ```tsx
   import { cn } from "@/lib/utils";
   
   export function CustomCard({ className, active }: { className?: string; active: boolean }) {
       return (
           <div className={cn("p-6 bg-secondary-dark border rounded-lg", active && "border-recording-red", className)}>
               Card Content
           </div>
       );
   }
   ```
3. **এইচটিএমএল সেমান্টিক ট্যাগ (Semantic HTML):**
   ডিজাইনের লেআউট এবং SEO উন্নত করতে যথোপযুক্ত সেমান্টিক ট্যাগ ব্যবহার করুন:
   - হেডার সেকশনের জন্য `<header>`
   - নেভিগেশন বারের জন্য `<nav>`
   - মূল কন্টেন্টের জন্য `<main>`
   - গুরুত্বপূর্ণ সেকশনের জন্য `<section>`
   - ফুটারের জন্য `<footer>`
